ZIMRA API Service

Welcome to the ZIMRA Fiscal Device Management System. Manage your fiscal devices and invoices with ease.

Manage Invoices API Status

Available API Endpoints:

GET /api/invoices - List all invoices
GET /api/invoices/{id} - Get invoice details
POST /api/submit_receipt/{device_id} - Submit receipt
POST /api/openday/{device_id} - Open fiscal day
POST /api/close_day/{device_id} - Close fiscal day
GET /api/getstatus/{device_id} - Get device status
